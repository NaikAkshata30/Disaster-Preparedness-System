# 🌍 Disaster Preparedness and Readiness Assessment System

A modern Django-based web application that helps users assess their disaster preparedness through interactive assessments, dynamic risk analysis, personalized recommendations, and real-time analytics dashboards.

## 🚀 Overview

The Disaster Preparedness and Readiness Assessment System is designed to improve emergency awareness and preparedness by evaluating user responses and generating meaningful preparedness insights.

The platform analyzes preparedness factors such as emergency planning, resource availability, communication readiness, and safety awareness to help users identify vulnerabilities and take proactive steps toward disaster readiness.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/NaikAkshata30/Disaster-Preparedness-System.git
cd Disaster-Preparedness-System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run Development Server

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
